exiftool.constants

This file defines constants which are used by other modules in the package

Module Contents

exiftool.constants.DEFAULT_BLOCK_SIZE :int = 4096

The default block size when reading from exiftool. The standard value should be fine, though other values might give better performance in some cases.

exiftool.constants.DEFAULT_EXECUTABLE :str

The name of the default executable to run.

exiftool (Linux) or exiftool.exe (Windows)

By default, the executable is searched for on one of the paths listed in the PATH environment variable. If it’s not on the PATH, a full path should be given to the ExifTool constructor.

exiftool.constants.DEFAULT_EXECUTABLE = exiftool.exe
